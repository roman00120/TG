# Total Ground (TG)

Este proyecto es la plataforma web oficial de **Total Ground**, enfocada en soluciones de ingeniería avanzada.

## Créditos
Desarrollado y mantenido por el **Ing. Román Velasco Moctezuma**.

## Descripción
Sitio web corporativo que incluye:
- Catálogo dinámico de productos.
- Sección de servicios especializados.
- Información sobre la empresa y eventos.
- Centro de recursos multimedia y manuales.

## Tecnologías
- HTML5
- CSS3 (Vanilla)
- JavaScript
