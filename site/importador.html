Importador Masivo de Productos

Usa esta herramienta para convertir tu lista de Excel en el código necesario para la web.

Paso 1: Prepara tu Excel

Organiza tu Excel con estas 4 columnas (en este orden) y sin encabezados:

  1. Nombre del Producto
  2. Categoría (ej: puesta-a-tierra, pararrayos, etc.)
  3. Precio (ej: Contactar, o 1500)
  4. Nombre del Archivo de Imagen (ej: foto1.jpg)
Categorías válidas: puesta-a-tierra, pararrayos, supresores, regulador-respaldo, telemetria, bot-tgone, banco-capacitores, infraestructura
